HISTORY

Tracing the evolution of Alfred Games from the first line of code to the latest AI-driven experiment.

Studio Inception

2023

The concept of Alfred Games was initialized. The goal: leverage emerging Generative AI tools to empower a solo developer to create assets and code at a scale previously impossible.

First Prototype

Early 2024

"Project: Genesis" (later scrapped) served as the testbed for the AI-assisted workflow. Lessons learned here about Unity integration and prompt engineering paved the way for future titles.

Coin Catcher 2D

Present

The first public release. A refined arcade experience demonstrating the viability of the tech stack. Features 100% AI-generated art and extensive AI-assisted C# scripting.

Dev Reflections

The AI Shift

Moving from traditional asset creation to generative pipelines reduced iteration time by approximately 80%. This allows for more focus on "game feel" rather than pixel pushing.

Code Architecture

Adopting a modular, component-based architecture in Unity proved crucial. It allows AI assistants to write self-contained scripts that plug into the larger system without breaking existing logic.
